# Employee Attendance Management System

Desktop application developed using C#, SQL Server, and a 3-Tier Architecture to manage employee attendance efficiently.

## Features

- Employee Management
- Attendance Registration (Check-In / Check-Out)
- Working Hours Calculation
- Leave & Absence Tracking
- Attendance Reports
- User Authentication & Role Management

## Architecture

- Presentation Layer (WinForms)
- Business Logic Layer (BLL)
- Data Access Layer (DAL)
- Entity Layer (BE)

## Technologies

- C#
- WinForms
- SQL Server
- LINQ
- Entity Framework
